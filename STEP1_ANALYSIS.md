# STEP 1 Analysis

## 기준
- 목표: `https://jvurglar-lgtm.github.io`에 반응형 개인 프로페셔널 웹사이트를 만들고, 상단 `Games` 메뉴와 지렁이 게임을 추가한다.
- 제약: 정적 `HTML`, `CSS`, `JavaScript`만 사용한다.
- 보안: 인증이 필요할 때만 `github_token.txt`의 토큰을 사용한다. 토큰은 이 문서, 코드, 로그, Git에 남기지 않는다.

## 확인된 입력
- GitHub Pages 주소: `https://jvurglar-lgtm.github.io`
- GitHub 저장소: `https://github.com/jvurglar-lgtm/jvurglar-lgtm.github.io.git`
- GitHub 토큰 파일: `github_token.txt`
- 프로필 참고 자료: 저장소에서 `CV.pdf`는 확인되지 않음
- 웹사이트 디자인 참고: 구체 자료 없음
- 게임 추가 기능: 구체 기능 미확정
