# CHANGE_REQUEST

## 원문 보존
- 프로필을 게임 개발자에 맞게 임의로 추가하여 작성, 전체 테마는 페이스북 스타일로 적용, 테트리스 미니게임을 추가

## 해석 요약
- 프로필 영역에 게임 개발자 콘셉트의 소개 문구를 추가한다.
- 전체 사이트 테마를 페이스북 스타일의 색감과 카드 구조로 바꾼다.
- 기존 Games 영역에 테트리스 미니게임을 추가한다.

## 중복·충돌·모호성
- 중복 요청 없음.
- C1과 C2는 서로 독립적이다.
- C3는 게임 로직이 큰 기능이라 완료 기준을 구현 전 최종 확정하는 편이 안전하다.
- "게임 개발자에 맞게 임의로"는 실제 경력 생성인지 콘셉트 문구인지 모호하므로 [사람 확인 필요]로 유지한다.
- "페이스북 스타일"은 구체 색상/레이아웃 기준이 없으므로 구현 시 디자인 참조를 추가로 확인하는 것이 좋다.
- 테트리스는 기존 지렁이 게임보다 범위가 크므로 모바일 조작, 입력 반응, 퍼즐 규칙을 먼저 확정하는 것이 좋다.

## 권장 순서
1. C1 프로필 문구 추가
2. C2 페이스북 스타일 테마 적용
3. C3 테트리스 미니게임 추가
